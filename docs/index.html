agentmon 🐱

Agent 小精灵 —— macOS 上的 Agent 长任务监控，用一只 AI 原创小猫陪你养成。

下载最新版 GitHub 源码

它做什么

监控本地 Agent 客户端（Claude Code 等）长任务的 启动 / 暂停 / 结束，用两种形态呈现：

🧭 菜单栏

实时显示每个客户端「工作中 / 等待中 / 已完成」数量，一眼掌握全局。

🐱 桌面宠物

小猫随你的工作状态积累/消耗能量，能量达门槛触发进化换肤。右键可隐藏，菜单栏再唤出。

🔒 本地 & 隐私

纯本地处理，无网络上报；日志只记事件元数据，绝不记录任务内容。

🩺 可自诊断

--doctor 与菜单「运行诊断」一键排查监控为何不工作。

能量玩法

事件能量变化（默认，可配置）
工作中任务+2 / 分钟
等待中任务−1 / 分钟
完成任务+30（一次性）
无任务−0.5 / 分钟

门槛 Lv2=300 / Lv3=900 / Lv4=2000，等级单调不回退；「已完成」按天清零。

快速开始

# 下载 Release 里的 agentmon.zip，解压后拖入「应用程序」，启动即可
# 首次启用集成后，请在 Claude Code 中【新开一个会话】，hooks 才会生效

# 从源码构建：
git clone https://github.com/xinyuehtx/agentmon.git && cd agentmon
swift build
.build/debug/agentmon --doctor   # 无 GUI 打印诊断报告

看不到监控信息？

1 · 运行诊断

菜单「运行诊断…」或 agentmon --doctor，检查集成/上报器/spool/状态。

2 · 看日志

菜单「打开日志文件」查看事件是否被摄取。

3 · 新开会话

最常见原因：启用集成后没有新开 Claude Code 会话。
